eDUcore MOO Core Manuals

The eDUcore manual collection is a set of Web documents containing the text of the online help available from within the eDUcore itself.  These Web documents may be easily browsed, printed, or used to preview the features of the eDUcore. The same text is available from within an eDUcore MOO itself, through the "help" command.

Like the eDUcore help databases, the manuals are organized functionally, primarily according to character class (user, builder, programmer, manager, and wizard).  Some accessory manuals (VSPO, editor, functions, core objects, and FUP) present supporting features or provide greater detail on special topics.

IMPORTANT! The eDUcore manuals, like the eDUcore itself, are based upon the LambdaMOO core from which they were originally derived and extended.  While the changes and additions Diversity University has made to these texts are copyright Diversity University, the original LambadMOO texts retain their original copyright.

The manuals are:

  • User Manual -  for all MOO users
  • Builder Manual -  for MOO builders
  • VSPO Manual - About the VSPO feature
  • Editor Manual - About the MOO editor
  • Programmer Manual - For MOO language programmers
  • MOO Language Functions Manual - Descriptions of the MOO language built-in functions
  • Core Objects Manual - About the eDUcore's basic objects
  • Manager Manual - For MOO Managers
  • FUP Manual - About the File Utilties Package system
  • Wizard Manual - For MOO Wizards
The complete set of these manuals is also available as a ZIP archive and as a tar.gz archive for downloading.

A separare set of documents for each of the programming utility libraries is currently in preparation.

If you have questions about the Diversity University eDUcore that aren't answered by the manuals, please also check the DU Forum devoted to the eDUcore.
