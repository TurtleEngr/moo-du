The MOO Server

In order to operate a MOO-based VR world, you need both a server and a database.  Nearly all MOOs operating today use the LambdaMOO Server, which was originally written for UNIX systems, but has also been compiled for the Win95/NT and Mac operating systems.  At this time, Diversity University only supports MOOs operating under UNIX and Win95 or WinNT, although support for the MacOS may be added in the future.

The material available here for MOOs running under UNIX is:

  • Setting Up the MOO's UNIX Directory Structure
  • Compiling the LambdaMOO Server for UNIX
  • Recommended LambdaMOO Server Patches
  • Recommended Supporting Scripts
The material available here to support MOOs running under the Win95/NT operating systems is:
  • Setting Up the LambdaMOO Server for Win95/NT
  • Recommended Supporting Scripts
